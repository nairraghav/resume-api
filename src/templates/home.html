Résumé API

This API has the following supports

User API

  • Create A User
  • Get All Users
  • Get A User By Id
  • Search For User By Email
  • Get User's Experience(s)
  • Update User By Id
  • Delete User By Id

Experience API

  • Create An Experience
  • Get All Experience
  • Get An Experience By Id
  • Update Experience By Id
  • Delete Experience By Id
