Résumé API

This API supports the following:

User API

  • Create A User [POST /api/user/create]

                        
    {
        "first_name":"{first_name}",
        "last_name":"{last_name}",
        "city_state":"{city_state}",
        "phone_number":"{phone_number}",
        "email_address":"{email_address}",
        "password":"{password}"
    }

  • Get All Users [GET /api/users]

  • Get A User By Id [GET /api/user/{user_id}]

  • Search For User By Email [GET /api/user/search?email={user_email}]

  • Get User Experience(s) [GET /api/user/{user_id}/experiences]

  • Update User By Id [PUT /api/user/{user_id}]

                        
    {
        "first_name":"{first_name}",
        "last_name":"{last_name}",
        "city_state":"{city_state}",
        "phone_number":"{phone_number}",
        "email_address":"{email_address}",
        "password":"{password}",
        "experiences": [{experience_id}]
    }

    Note: All fields are optional

  • Delete User By Id [DELETE /api/user/{user_id}]

Experience API

  • Create An Experience [POST /api/experience/create]

                        
    {
        "company_name": "{company_name}",
        "position_title":"{position_title}",
        "start_date":"{mm/dd/yyyy}",
        "end_date":"{mm/dd/yyyy}",
        "description":"{description}"
    }

  • Get All Experience(s) [GET /api/experiences]

  • Get An Experience By Id[GET /api/experience/{experience_id}]

  • Update Experience By Id[PUT /api/experience/{experience_id}]

                        
    {
        "company_name": "{company_name}",
        "position_title":"{position_title}",
        "start_date":"{mm/dd/yyyy}",
        "end_date":"{mm/dd/yyyy}",
        "description":"{description}"
    }

    Note: All fields are optional

  • Delete Experience By Id[DELETE /api/experience/{experience_id}]
